# 🛒 E-Commerce React Assignment

## 📌 Project Overview
This is a basic e-commerce web application where users can:
- Browse products
- View product details
- filter product according to category
- Add items to cart

---

## 🚀 Tech Stack
- React JS 
- React Router
- Context API
- typescript
- FakeStore API

Testing
Playwright/Cypress used for basic E2E testing
